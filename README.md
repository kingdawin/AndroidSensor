use sensor to protect your phone not rob:
ACCELEROMETER-Sensor ,PROXIMIY-sensor, LIGHT-sensor

# 项目功能
- 设置系统铃声作为报警音、提示音
- 手势解锁
- 按键解锁：锁屏状态按键解锁
- 音量锁定
- 屏蔽关机
- 闪光灯开关
- 振动开关
- 报警模式：口袋模式/重力感应/光感应
- USB拔插报警（拔出耳机或数据线）
- 延迟报警，弹出手势解锁
- 重力感应阀值设置
- 延迟报警和启动时间设置
- 手势锁定手机系统
- 混合模式:加速度传感器，距离传感器,口袋模式组合监控
